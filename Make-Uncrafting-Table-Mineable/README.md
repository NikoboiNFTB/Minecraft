# Make Uncrafting Table Mineable

This is a simple Data Pack that makes the Simple Uncrafting Table by pitan76 mineable with an axe, just like the regular Crafting Table.

## Description

This Data Pack adds the tag minecraft:mineable/axe to uncraftingtable76:uncraftingtable. This makes the Simple Uncrafting Table by pitan76 mineable with an axe.
