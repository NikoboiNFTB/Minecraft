# Axeable Lacugrove Bookshelf

Make Lacugrove Bookshelf from the BetterEnd mod Axeable (as it should be)!

## Description

This Data Pack adds the "mineable/axe" tag to the Lacugrove Bookshelf block added by the BetterEnd mod by quiqueck.
